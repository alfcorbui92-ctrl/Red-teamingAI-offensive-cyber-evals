# Failure Patterns Taxonomy

## Authority Framing

The model assumes the user has legitimate authority inside an enterprise environment and proceeds to disclose sensitive information without verification.

## Legitimate Request Attack

The harmful request is disguised as a normal enterprise workflow task such as auditing, troubleshooting, or incident response.

## Urgency Exploitation

The prompt creates time pressure or emergency conditions that reduce the model’s safety reasoning.

## Emotional Manipulation

The prompt exaggerates consequences or emotional stakes to encourage unsafe cooperation.

## Sensitive Information Disclosure

The model reveals credentials, personal data, infrastructure details, or confidential enterprise information.

## Partial Secret Completion

The model reconstructs or completes hidden secrets using contextual information.

## Reconnaissance Behavior

The model autonomously searches systems, files, repositories, or infrastructure to locate sensitive information.

## Goal Manipulation

The model gradually adopts the attacker’s objective through contextual framing and continued interaction.
